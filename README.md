## 🌟 Overview

**VARUNO** is a modern fashion e-commerce platform that delivers a seamless shopping experience. From curated collections to exclusive inner circle access, VARUNO combines style with soul. The platform features a robust authentication system, newsletter integration, contact support, and a comprehensive admin panel.

### 🎯 Key Highlights
- 🛍️ **Curated Fashion Collections** - Timeless classics meets bold trends
- 💌 **Inner Circle Newsletter** - Exclusive drops and style guides
- 🔐 **Secure Authentication** - JWT-based user authentication
- 📧 **Automated Email System** - OTP verification, welcome emails, and notifications
- 👑 **Powerful Admin Panel** - Complete control over products and users
- 📱 **Fully Responsive** - Flawless experience on all devices

## ✨ Features

### 🛒 **User Features**
- User registration & login with email verification
- Browse fashion collections
- Product search and filtering
- Add to cart and wishlist
- Secure checkout process
- Order tracking
- Newsletter subscription
- Contact support

### 👑 **Admin Features**
- Dashboard with analytics
- Product management (CRUD operations)
- Inventory management
- Order management
- User management
- Newsletter subscriber management
- Email template customization

### 📧 **Email Features**
- OTP verification emails
- Welcome emails for new users
- Newsletter welcome emails
- Contact form auto-replies
- Admin notifications
- Order confirmation emails
- Password reset emails
